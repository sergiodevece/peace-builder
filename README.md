# Peace Builder

Videojuego arcade tipo Space Invaders en el que el jugador convierte monstruos en ángeles para construir un mundo mejor.

## Objetivo del proyecto

El objetivo de este proyecto es practicar lógica de juego, interacción en navegador y diseño de una experiencia arcade con un mensaje positivo.

La propuesta toma una referencia clásica de los videojuegos arcade y le da la vuelta: en lugar de destruir enemigos, la mecánica se orienta a transformar el conflicto en algo mejor.

## Características principales

- Videojuego arcade inspirado en Space Invaders.
- Mecánica de transformación de monstruos en ángeles.
- Enfoque visual y narrativo positivo.
- Proyecto ligero orientado a navegador.
- Experiencia sencilla pensada para aprendizaje y experimentación creativa.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto funcional en fase inicial de aprendizaje y mejora continua.

## Aprendizajes trabajados

Con este proyecto se practican conceptos como:

- Estructura básica de un videojuego web.
- Control de eventos e interacción con el usuario.
- Lógica de enemigos, objetivos o puntuación.
- Adaptación creativa de mecánicas arcade clásicas.
- Uso de JavaScript para construir experiencias interactivas.
